계층(색상)

계층 구조

Primary 화면의 주요 목표를 달성하는 가장 중요한 액션 버튼 (화면당 1개 권장)
예: 등록, 저장, 제출, 완료
Secondary Primary를 보조하는 중요 기능으로 아웃라인 스타일 적용
예: 이전, 임시저장, 미리보기
Tertiary 자주 사용되지 않는 부가 기능으로 시각적 강조 최소화
예: 관리, 추가, 수정, 초기화, 다운로드, 취소, 닫기
Inverse 고대비 반전 스타일, 배경과 명확한 구분이 필요할 때 사용
예: 조회하기, 검색, 필터 적용
Subtle 거의 눈에 띄지 않는 최소 UI로 보조 링크나 비중요 액션에 사용
예: 더보기, 상세보기, 외부 링크

CSS 커스텀 셀렉터

마크업 가이드, CSS 선택자에 대한 표는 필수, 선택의 구성으로 이루어져 있으며, 세부 항목으로는 전체영역, 색상, primary, secondary, tertiary, 사이즈, 더 작게, 작게, 중간, 크게, 더 크게, 텍스트형, 아이콘만 있는 버튼, 기본, border 및 bg 항목을 포함한다.
필수 클래스 선택 클래스
전체영역 .btn_base 사이즈 .sm 작게
.md 중간
.lg 크게
.xl 더 크게
계층(색상) .primary 화면의 핵심 목표 달성 (화면당 1개 권장)
.secondary Primary 보조, 자주 사용되는 중요 기능
.tertiary 가끔 사용하는 부가 기능, 시각적 강조 최소화
.inverse 어두운 배경색 버튼, 시각적 대비가 필요한 영역에 사용
.subtle 거의 보이지 않는 최소 강조 (외부 링크, 비중요 액션)
텍스트형 .ghost 기본
.link 링크
아이콘만 있는 버튼 .icon_only 정사각형 형태

탭 | tab

            
 탭(Tab)은 관련된 콘텐츠를 그룹화하여 탭 전환으로 표시하는 데 사용하며, 활성화된 탭을 시각적으로 강조합니다.

                코드 보기
                
              

              

                

                  
<section class="tab_wrap w-50">
    <ul class="tab_list">
      <!-- 활성화 버튼에 active 추가 -->
      <li data-tab="tab01" class="tab_item active"><a href="#" class="inner" onclick="return false">1번째 탭</a></li>
      <li data-tab="tab02" class="tab_item"><a href="#" class="inner" onclick="return false">2번째 탭</a></li>
    </ul>

    <!-- 탭이면 tab_panel추가 /활성화 탭에 active 추가 -->
    <div class="tab_panel active" id="tab01">1번</div>
    <div class="tab_panel" id="tab02">2번</div>
  </section>

                

              

            

          

        


        
        

          

            
카드 | Card

            
카드(Card)는 관련된 정보를 하나의 독립적인 영역으로 그룹화하여 표시하는 데 사용하며, 제목, 날짜, 뱃지 등을 포함할 수 있습니다.
